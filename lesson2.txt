خطوات رفع مشروع إلى GitHub:
تهيئة المستودع المحلي:
افتح موجه الأوامر (Terminal أو Git Bash) في مجلد مشروعك المحلي.
اكتب الأمر التالي لبدء تهيئة Git في مجلدك:
bash
git init
إضافة الملفات إلى مرحلة التخزين المؤقت:
استخدم الأمر التالي لإضافة جميع الملفات والتغييرات إلى مرحلة التخزين المؤقت:
bash
git add .
تسجيل التغييرات (Commit):
اكتب الأمر التالي لإنشاء سجل (commit) للتغييرات التي قمت بها:
bash
git commit -m "رسالة تصف التغييرات"
(استبدل "رسالة تصف التغييرات" برسالة مختصرة تعبر عن طبيعة التغييرات)
إنشاء مستودع بعيد على GitHub:
اذهب إلى GitHub وأنشئ مستودعًا جديدًا (New repository).
انسخ رابط HTTPS للمستودع الجديد الذي أنشأته.
ربط المستودع المحلي بالمستودع البعيد:
استخدم الأمر التالي لاستبدال <رابط_المستودع> بالرابط الذي نسخته من GitHub:
bash
git remote add origin <رابط_المستودع>
رفع التغييرات إلى GitHub:
اكتب الأمر التالي لرفع التغييرات إلى فرع main في المستودع البعيد:
bash
git push -u origin main
سيطلب منك GitHub تسجيل الدخول باستخدام اسم المستخدم وكلمة المرور الخاصين بك إذا لم تكن مسجلاً دخولك بالفعل
